然后跳到_init_函数，categories字典存的是指纹类型和对应优先级，technologies存的是不同组件的指纹，根据wappalyzer在github上的readme，一些字段的定义如下：

必须属性：

字段 描述
cats 一个或多个category ID
website 应用网站的URL

可选属性：

字段 描述
description 对该项技术的简单描述
icon 应用的icon文件名
cpe CPE 是一种结构化的技术命名方案
saas 布尔值，表示该项技术是否是软件即服务(SaaS)，也就是托管或基于云的
oss 技术有开源认证
pricing 技术的价格，花费(字面意思)
implies 一种应用的出现意味着另一种技术的出现，比如WordPress意味着PHP
requires 与implies类似，但仅在识别出所需技术时才运行检测。适用于特定 CMS 的主题。
requiresCategory 类似于requires;仅当已识别出所需类别中的技术时，才会运行检测。
excludes 与implies相反。一个应用程序的存在可以排除另一个应用程序的存在。
